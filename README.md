# meeting-services
该项目是一个完全开源的在线视频会议服务，使用纯Go编写。包括SFU服务和会管服务，基于livekit/pion二次开发，提供了更丰富的会议功能，类似于腾讯会议。 This project is a fully open-source video conferencing service written in pure Go. It includes SFU service and conference management service, based on livekit/pion with additional richer conferencing features, similar to Tencent Meeting.
